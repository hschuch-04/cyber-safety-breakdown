## Introduction
### User
**The Cyber Safety Toolkit** is a web application where users can visit to learn the best and current practices with keeping your online information and devices safe from security threats. This web application allows users to create a profile, do a quiz, view articles, view guides, and a FAQ.

**The Quiz** is an optional feature of the application which allows users to take a short questionnaire about general internet security questions and concepts. They will then be given a result which will then allow for the Toolkit to give recommendations for articles and guides based on their skill level. The quiz can be retaken at a later date if the user wants.

**Articles** are designed to give information to the user about current practices or any upcoming technology advancements that general population may want to learn about or relevant towards. This is a completely education section of the application, and their will be a guide about what kind of information will be conveyed and the knowledge level of the article.

<img src="https://github.com/hschuch-04/qps-application/assets/46430331/46200f7d-8a56-4541-a57b-704db047d1c8" width="640" height="420" />

**Guides** are designed to take what the user has learnt and put it into practice, giving a step-by-step guide on how to implement certain daily securities features which can make a massive amount of difference. This will include pictures, and a knowledge/difficulty level.

**The FAQ** allowed users to send questions through directly to admins which allowed administration to respond to their questions. These will be updated in real-time when the admin responds to the question.


### Administration
The Toolkit also supports an admin portal which gives them the ability to edit/update/delete user data, profiles, FAQs, etc. It also has a metrics system which will indicate the general age of people using the application, along with the general knowledge level displayed in graphs.

<img src="https://github.com/hschuch-04/qps-application/assets/46430331/fa465878-e9af-4a5f-9699-87a3a0741472" width="640" height="360" />

The main part of this is the ability to create guides, and articles directly from the administration portal, so no programming is required. These can be hidden, displayed or deleted whenever by an administrator and will update in real-time requiring no restart of the web app. This was done to allow people without programming knowledge within the company to write up a security article/guide and reduce the overhead required to update.
